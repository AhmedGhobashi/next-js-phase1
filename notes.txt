1- why caching matters?
  - cache is like.. تخزين مؤقت
  - the flow [client requests data from server (let's say.. requests posts api), 
    the api sends the data to nextjs server, the next js server takes a cache from the data "until we need it" ]

2- in the posts page we will cancel caching, because sometimes the posts change, 
   and we don't want to get old data from the cache that are not updated yet.
